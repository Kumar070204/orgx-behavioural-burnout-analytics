## Problem Statement Chosen
Early Detection of Student Burnout & Dropout Risk

## Problem Understanding
Universities often detect academic issues only after performance declines. 
This project aims to build a behavioural analytics system that predicts:
- Burnout Risk Level
- Academic Disengagement Indicators
- Dropout Probability

## Proposed Approach
We simulate behavioural data representing:
- LMS login frequency
- Assignment submission delays
- Attendance trends
- Sentiment scores from feedback

A risk scoring model will be developed using weighted behavioural indicators.

## Dataset
Type: Synthetic  
Reason: No real behavioural dataset available  
Records: Planned 1000+ simulated students  

## Feature Engineering Plan
- Normalize behavioural indicators
- Create composite burnout risk score (0–100)
- Categorize risk levels (Low / Medium / High)

## Expected Outputs
- Risk Score
- Risk Classification
- Key behavioural triggers
- Recommended intervention strategies
